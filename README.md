# Data Structures & Algorithms (DSA)

This repository is a **topic-wise learning guide** for Data Structures & Algorithms, built to help people *understand* concepts (not just memorize solutions).

Each topic lives in its own directory and includes:
- `notes.md` — what you need to learn for that topic (concepts, patterns, tricks, common mistakes)
- example / practice questions — to apply what you learned from the notes

I’m adding more topics over time.

---

## How this repo is organized

Each folder represents a DSA topic. Inside you’ll typically find:

- `notes.md`
- practice questions / examples (file names may vary)

Example:
- `Arrays & Hashing/`
  - `notes.md`
  - (questions / examples)

## How to use this repository

1. Start with a topic folder.
2. Read `notes.md` and make sure you understand the concepts.
3. Attempt the example questions without looking up solutions.
4. Revise your approach, then re-try similar problems.
5. (Optional) Add your own notes/learnings to `notes.md`.
